# ICar


Quickstart
----------
for running the server locally
- first install 
```bash
pip fastapi uvicorn sqlalchemy sqlalchemy_utils psycopg2 python-dotenv
```
- run the local server with localhost:8080
```bash
uvicorn main:app
```
